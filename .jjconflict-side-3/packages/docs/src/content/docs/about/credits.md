This nix infrastructure config builds on a variety of other work from the Nix community.

## Primary frameworks

### flake-parts ecosystem

- [hercules-ci/flake-parts](https://github.com/hercules-ci/flake-parts) - Modular flake composition framework
- [srid/nixos-unified](https://github.com/srid/nixos-unified) - Directory-based autowiring for flake-parts
- [srid/nixos-config](https://github.com/srid/nixos-config) - Reference implementation and patterns

### Multi-channel resilience

- [mirkolenz/nixos](https://github.com/mirkolenz/nixos) - Multi-channel nixpkgs patterns and overlay composition strategies

### Other inspirations

- [ehllie/dotfiles](https://github.com/ehllie/dotfiles) - Flake-parts integration patterns
- [NickCao/flakes](https://github.com/NickCao/flakes) - Advanced flake techniques
- [EmergentMind/nix-config](https://github.com/EmergentMind/nix-config) - Practical nix-config organization
- [wegank/nixos-config](https://github.com/wegank/nixos-config) - NixOS configuration examples
- [MatthiasBenaets/nixos-config](https://github.com/MatthiasBenaets/nixos-config) - Multi-system configuration patterns
- [Misterio77/nix-config](https://github.com/Misterio77/nix-config) - Comprehensive nix-config reference

## Core technologies

- [Nix](https://nixos.org/) - Reproducible builds and deployments
- [NixOS](https://nixos.org/) - Linux distribution built on Nix
- [nix-darwin](https://github.com/LnL7/nix-darwin) - Nix-based macOS system configuration
- [home-manager](https://github.com/nix-community/home-manager) - User environment management with Nix
- [sops-nix](https://github.com/Mic92/sops-nix) - Secrets management for Nix
