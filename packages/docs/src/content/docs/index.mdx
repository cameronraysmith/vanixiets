## Quick start

```bash
# Clone and setup
git clone https://github.com/cameronraysmith/nix-config.git
cd nix-config
nix develop
bun install

# Start development
just docs-dev
```

See [Getting started guide](/guides/getting-started/) for complete setup instructions.

## Infrastructure documentation

This site documents the infra repository:

1. **System configurations** - NixOS and nix-darwin configurations
2. **Infrastructure patterns** - Reproducible development environments and deployment workflows
3. **Kubernetes deployments** - GitOps with nixidy and holos
