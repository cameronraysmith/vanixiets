# openspec vendored claude assets

This directory holds vendored OpenSpec (`@fission-ai/openspec`) Claude assets generated by `regen.sh`.
The `assets/skills/` and `assets/commands/opsx/` trees are committed generated output, produced by running `openspec init --tools claude` against the version pinned by the `llm-agents` flake input.
They are injected at the user level for crs58 via `aiSkills.extraSkillDirs` and `programs.claude-code.commandsDir` in `modules/home/users/crs58/default.nix`.
The assets are version-portable; the only dynamic field is the `generatedBy` frontmatter line on each `SKILL.md`.
Do not hand-edit these files; regenerate via `regen.sh` (or `just openspec-regen`) after an `llm-agents` input bump instead.
